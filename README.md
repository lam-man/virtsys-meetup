# virtsys-meetup
Virtualized Systems Meetup 2024

## Annoucement 
Thanks for visiting SPLVM "Vitual System Meetup". We will follow UCSD CSE291 on Virtualization. Please check our schedule below and join us if you are interested. The meetup will be held every Monday at 5:00pm PST. The first meetup will be on January 22, 2021. Please contact us if you want to present a paper or give a guest talk. 

## Important Links
- UCSD CSE291: [Virtualization](https://cseweb.ucsd.edu/~yiying/cse291-winter22/reading/)
